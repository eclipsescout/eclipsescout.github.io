Outline

This document is referring to a past Scout release. Please click here for the recent version.

The outline is a special variant of the tree widget mainly used in the navigation area of the Desktop.

It contains Pages which are a special variant of tree nodes.

The desktop can contain multiple outlines but only one outline can be active at a time.
